Building the PLC server

Required software
-JDK 1.6
-Apache Maven

From the same directory as this README file run
-'mvn package'
-the build outputs a zip file into the target directory

Project file genaration
-'mvn eclipse:eclipse'
-eclipse will need the ScalaIDE for Eclipse plugin available here: http://scala-ide.org/download/nightly.html
-import the project include Eclipse as an existing project

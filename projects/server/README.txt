Running the server

Required softare
-JDK 1.6
-Play Framework 2.0

Configuration
-conf/application.conf allows the address and port of the plc server to be set

From the same directory as this README file run
-'play run'
-Open a web-browser to localhost:9000

Project file genaration
-'play eclipsify'
-import the project include Eclipse as an existing project
